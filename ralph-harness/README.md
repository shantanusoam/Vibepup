# Ralph Harness Demo

A simple demonstration project for the Ralph autonomous coding agent harness.

## Project Structure

- `helloworld.py` - Main script that prints a greeting message
- `test_helloworld.py` - Unit tests for the helloworld module

## How to Run

### Run the main script:
```bash
python3 helloworld.py
```

### Run the tests:
```bash
python3 test_helloworld.py
```

Expected output from main script:
```
Hello from Ralph
```
